❓ Help & Guide

Purpose

The most important section. Every mistake you record and review is a future mark saved. Track wrong thinking, correct concept and lesson learned.

How to Add a Mistake

  • Select Subject and Chapter.
  • Choose Mistake Type: Concept / Formula / Calculation / Careless / Other.
  • Write the Question you got wrong (or a short description).
  • Write your Wrong Thinking â€” exactly what you thought incorrectly.
  • Write the Correct Concept â€” the right approach or answer.
  • Write the Lesson Learned â€” one sentence to remember.
  • Set Status: Pending (not yet revised) or Revised.
  • Set a Revision Date to be reminded to revisit this mistake.
  • Click Save.

Status Workflow

Mark a mistake as Revised only after you can solve a similar question correctly without any help.

Summary

The summary shows breakdown by mistake type. If Calculation is high, practise slower careful working. If Concept is high, revisit theory.

Tip: Read your Mistake Book 2 days before any test. It is the fastest way to avoid repeating errors.

What is Study Companion?

A complete offline JEE/NEET study management app. All data is stored locally in your browser â€” no internet, no login, no cloud.

Important: Always use the same browser on the same device. Clearing browser data will delete all records. Use the Export button on the Dashboard regularly.

Backup & Restore

  • Export All Data (Dashboard) → saves a JSON file with all records from every section.
  • Export CSV (on each page) → opens in Excel / Google Sheets.
  • Import Data (Dashboard) → select a previously exported JSON to restore data. Records are added, not overwritten.
Tip: Export your data every week. Store the backup file in Google Drive or email it to yourself.
