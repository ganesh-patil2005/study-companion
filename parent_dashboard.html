❓ Help & Guide

Purpose

A dedicated log for parents to record daily observations, encouragement and study discussions. Parental involvement is a proven factor in student performance.

How to Use

  • Set today’s Date.
  • Write a Positive Observation about the student’s effort or behaviour.
  • Log what Appreciation was given that day.
  • Note whether a meaningful Study Discussion happened (even 10 minutes).
  • Add any general Notes or encouragement.
  • Click Save.

Reviewing Entries

Past entries appear in the table below. Reading them together with the student builds trust and shows that effort is being noticed.

Tip: Focus on effort, not just results. “I noticed you studied for 3 hours today” is more motivating than focusing only on scores.

Backup & Restore

  • Export All Data (Dashboard) → saves a JSON file with all records from every section.
  • Export CSV (on each page) → opens in Excel / Google Sheets.
  • Import Data (Dashboard) → select a previously exported JSON to restore data. Records are added, not overwritten.
Tip: Export your data every week. Store the backup file in Google Drive or email it to yourself.
