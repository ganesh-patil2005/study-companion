❓ Help & Guide

Dashboard Overview

The Dashboard shows a live count of records in each section. Click any card to open that section.

Today’s Revisions Alert

  • The Revision Plans card shows a red badge if you have revisions scheduled for today that are not yet Completed.
  • The subjects and chapters due today are listed inside the card.

Export All Data

  • Click Export All Data to download a complete JSON backup of every section including Daily Planner data.
  • Type a device name (e.g. “Phone” or “Laptop”) in the label box next to the button — the filename becomes StudyCompanion_Phone_2026-07-19.json so you know which device it came from.

Merge Import (multi-device sync)

  • Click Merge Import and select a JSON file exported from another device.
  • Only records not already present on this device are added. Duplicates are detected automatically and skipped.
  • A section-by-section summary shows how many records were added vs skipped.
  • Your existing data is never deleted or overwritten during a merge.
Tip for multi-device use: Export from Device A → send the JSON file via WhatsApp / Google Drive / email to Device B → open the app on Device B → click Merge Import. Repeat in reverse to bring Device B’s new data to Device A.

What is Study Companion?

A complete offline JEE/NEET study management app. All data is stored locally in your browser — no internet, no login, no cloud.

Important: Always use the same browser on the same device. Clearing browser data will delete all records. Use the Export button on the Dashboard regularly.

Pages at a Glance

📘 Learning Book

Record concepts, theories and key notes topic-wise. Your personal textbook.

✏ Practice Book

Log every practice session — questions, accuracy, time and remarks per chapter.

⭐ Mistake Book

Track every mistake with wrong thinking vs correct concept. Review before exams.

📗 Formula Book

Build your own formula sheet with memory tricks. Use the symbol toolbar to insert math symbols.

📙 Success Journal

Daily reflection — achievements, challenges, goals. Builds consistency and motivation.

🔁 Revision Planner

Schedule and track revision sessions. Dashboard alerts you of today’s pending revisions.

📊 Dashboard

Overview of all records, today’s revision alert, and backup (Export/Import).

👨 Parent Dashboard

Parents log daily observations, encouragement and study discussion notes.

Subject & Chapter Dropdowns

  • Pre-loaded with the full JEE/NEET syllabus for Physics, Chemistry, Mathematics and Biology.
  • Select “+ Add New Subject…” or “+ Add New Chapter…” to add custom entries — they are saved permanently.
  • Topics use a free-text field with autocomplete suggestions.

Backup & Restore

  • Export All Data (Dashboard) → saves a JSON file with all records from every section. Enter a device label first so the filename identifies the source device.
  • Export CSV (on each page) → opens in Excel / Google Sheets.
  • Merge Import (Dashboard) → select a JSON exported from another device. Only NEW records are added; duplicates automatically skipped. Shows a summary of what was merged.
Tip: Export your data every week. Store the backup file in Google Drive or email it to yourself.
