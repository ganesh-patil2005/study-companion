❓ Help & Guide

Purpose

Record concepts, theories and key formulas in your own words ├бРѓгРђЮ topic by topic. This becomes your personal textbook for revision.

How to Add a Note

  • Select Subject from the dropdown (Physics, Chemistry, Maths, Biology).
  • Select Chapter ├бРѓгРђЮ the list filters automatically based on subject.
  • Type or select a Topic (autocomplete suggestions appear).
  • Write My Understanding in your own words ├бРѓгРђЮ this is the most effective way to retain concepts.
  • Add key Formulas for that topic.
  • Click Save.

Editing & Deleting

  • Click Edit on any row to load it back into the form. Make changes and Save again.
  • Click Delete to permanently remove a note.

Search

Use the search box to filter notes by subject, chapter, topic or any keyword in your notes.

Tip: Write notes immediately after studying a topic. Do not copy from textbooks ├бРѓгРђЮ use your own words for better retention.

What is Study Companion?

A complete offline JEE/NEET study management app. All data is stored locally in your browser ├бРѓгРђЮ no internet, no login, no cloud.

Important: Always use the same browser on the same device. Clearing browser data will delete all records. Use the Export button on the Dashboard regularly.

  • Pre-loaded with the full JEE/NEET syllabus for Physics, Chemistry, Mathematics and Biology.
  • Select “+ Add New Subject…” or “+ Add New Chapter…” to add custom entries ├бРѓгРђЮ they are saved permanently.
  • Topics use a free-text field with autocomplete suggestions.

Backup & Restore

  • Export All Data (Dashboard) → saves a JSON file with all records from every section.
  • Export CSV (on each page) → opens in Excel / Google Sheets.
  • Import Data (Dashboard) → select a previously exported JSON to restore data. Records are added, not overwritten.
Tip: Export your data every week. Store the backup file in Google Drive or email it to yourself.
